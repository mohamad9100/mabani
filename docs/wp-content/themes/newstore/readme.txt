== Description ==

NewStore is WordPress theme that help you start sale online. it is the perfect free theme designed for your WooCommerce shop based on bootstrap. it support WooCommerce shop. install recommended plugns and you will get awesome store. you will be able to use it for clothing, electronics, Mobile, foods and restaurants Store, interior store, gadgets store, home appliances shop, automobile,  books store, photo store, movies store, jewellery and almost any kind of store. nicely designed FromPage template for storefront. This theme supports popular plugins like Elementor Page Builder, Contact Form 7, Jetpack by WordPress.com, One Click Demo Import, WooCommerce and many more.  There are four demos available to choose, you may use any one you like and make changes to your website. View demo site at https://demo.themefarmer.com/newstore/newstore-demos/

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Frequently Asked Questions ==

= How to setup FrontPage ? =
1. create a page "Home" Go to Pages -> All pages -> Add New name it "Home" select front page template.
2. create a page "Blog" Go to Pages -> All pages -> Add New name it "Blog".
3. go to Settings -> Reading -> Your homepage displays  -> A static page -> select "Home" for Homepage select "Blog" for Post Page -> save changes.
4. go to Appearance -> Customize -> Frontpage Sections -> 
    * banners -> select image and enter links.
    * Slider -> Add New Slide.
    * Featured Categories -> select Categories.
    * FrontPage Top Widget Area. add "TF Product" Widget -> enable carousel, hide product info carousel count 7. 
    * FrontPage Products Widget Area. add "TF Product" Widget.
    * FrontPage Products Widget Area Column add four "Product" Widget.


= How to setup Mega Menu ? =
1.Go to Appearance > Menu. create a menu select location primary. -> add items enable switch "Mega Menu" for 0 depth item. 1 depth item will work as columns.
2. you may select background image for 0 depth item and image for grater than 1 depth items.
3. same way setup Product Menu.

= Does this theme support any plugins? =

NewSrore includes support for Infinite Scroll in Jetpack.
